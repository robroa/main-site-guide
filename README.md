# main-site-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不针对任何特定域名或具体网站，仅供整理、查阅和后续维护使用。

## 项目简介

这是一个通用型的 HTML 页面归档仓库。仓库中可能包含不同类型的演示页面、说明文档、工具界面或存档内容。每个 HTML 文件都是独立的，可以单独打开浏览，无需依赖后端服务。

## 目录结构

仓库目录结构建议如下：

```
main-site-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
├── docs/           # 相关文档或说明文件
└── README.md       # 本文件
```

你可以根据需要调整目录结构，但建议保持页面文件集中存放，便于管理。

## 页面归档说明

- 每个 HTML 页面应尽量自包含，避免跨文件依赖。
- 如需引用外部资源（如样式、脚本），请将资源文件放入 `assets/` 目录，并使用相对路径引用。
- 页面文件名建议使用英文小写，以 `-` 或 `_` 分隔单词，避免特殊字符。
- 可在页面内添加注释或说明，标明页面用途、创建日期、作者信息等。

## 维护说明

- 本仓库由维护者定期检查页面可用性，但不对页面内容做持续更新承诺。
- 欢迎提交 Issue 或 Pull Request 来报告问题、提出改进建议或添加新的归档页面。
- 添加新页面时，请尽量保持与现有页面风格一致，并更新本 README 文件（如需要）。
- 历史版本建议通过 Git 标签或分支进行标记，便于回溯。
